# Software Dev Workflow Skill

中文软件开发工作流 Skill，用于根据项目形态、开发阶段和当前场景判断下一步该做什么。

## Skill

- Skill 目录：`software-dev-workflow/`
- 入口文件：`software-dev-workflow/SKILL.md`
- 主要能力：spec-driven 开发、项目形态选择、阶段判断、starter blueprint、design doc 模板、验证门禁、PR readiness。

## Install

复制或安装 `software-dev-workflow/` 到 Codex skills 目录：

```bash
cp -R software-dev-workflow ~/.codex/skills/
```

然后在新会话中使用 `$software-dev-workflow`。
